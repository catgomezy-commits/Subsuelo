# Subsuelo — Sustainable Fashion Platform

> _Fashion that regenerates_

A digital platform for sustainable fashion, built with an organic futuristic aesthetic and interactive mouse-driven experiences.

## ✦ Sections

- **Upcycling Collection** — Zero-waste pieces made from deadstock fabrics, vintage garments, and industrial offcuts
- **Limited Edition** — Exclusive numbered drops with artisanal craftsmanship
- **Emerging Designers** — A showcase of global talent shaping fashion's future
- **Experiences & Events** — Mending circles, runway shows, material labs, and swap symposiums

## ✦ Features

- Custom cursor with organic glow effects
- Real-time particle system that responds to mouse movement
- 3D tilt cards on collection and designer items
- Mouse-driven parallax on headings
- Scroll-triggered fade-in animations
- Fully responsive design
- Dark, organic-futuristic visual language

## ✦ Tech Stack

HTML · CSS · JavaScript (vanilla)

Canvas API for ambient particle background
Intersection Observer for scroll reveals
CSS Grid & Flexbox for layout

## ✦ Getting Started

Clone the repo and open `index.html` in any browser:

```bash
git clone git@github.com:catgomezy-commits/Subsuelo.git
cd Subsuelo
open index.html
```
